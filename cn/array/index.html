Array包扩展

Array包扩展

对JavaScript中的Array做了扩展。

tango.linear.array.zip(arr1, arr2): []

返回一个新的数组，将两个数组arr1和arr2并列合并。

Array.zip([1, 2, 3], ['A', 'B', 'C'])
// [[1, 'A'], [2, 'B'], [3, 'C']]

tango.linear.array.swap(arr, i, j): void

交换数组中两个索引iii和jjj的值，该操作直接改变数组。

.clone(): []

使用[...arr]。

.zip(that): []

返回一个新的数组，将该数组和that数组并列合并。

请使用tango.linear.array.zip(this, that)。
